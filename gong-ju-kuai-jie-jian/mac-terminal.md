# mac terminal

1.option + click——可以將游標移到點擊位置

2.command + T——開啟新的當前視窗分頁

3.command + shift + N——開啟新視窗分頁

4.command + L——清除之前記錄

5.command + W——關閉當前分頁

6.control + tab / control + shift + tab——切換分頁

7.control + U——刪除游標的前面整行

8.control + K——刪除游標的後面整行

9.control + A——移動游標到最前面

10.control + E——移動游標到最後面

11.command + Q——刪除應用
